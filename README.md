# yakyu-coach-ai

## サービス概要
野球中継の専門用語に置いていかれたり、子どもの試合で何が起きているか分からない人のためのアプリです。
AIが**あなたの理解度**に合わせて、専門用語を野球以外の例えや例観戦時のポイントを使って分かりやすく解説します。
検索するだけでその場で理解でき、恥ずかしくて聞けない疑問もすぐに解消できます。

## このサービスへの思い・作りたい理由
高校時代、野球部の保護者会に父が参加しましたが、周りは野球経験者や指導者ばかりで、野球のルールしか分からない父は戦術や専門用語の話についていけず、あまり会話に参加できなかったみたいです。当時は「仕方ない」と思っていましたが、自分が社会人になり野球未経験者と話す中で、野球に興味はあるのに理解できず楽しめない人が多いと気づきました。野球観戦は好きだけど、詳しい話はわからない人、子どもが野球を始めて、もっと理解したいと思っている保護者、野球に興味はあるけど、どこから学べばいいかわからない人でも理解度に合わせて学べるアプリがあれば、もっと野球を楽しめるのではと思い、開発を決めました。

## ユーザー層について
### 野球初心者
- パートナーが野球好きで一緒に観戦したい人
- テレビで野球を見始めたけど用語が分からない人
- 職場の先輩や後輩と野球の話についていけない人
- 野球部のマネージャー

### 子供が野球を始めた保護者
- 子供が部活やリトルリーグに入った母親・父親
- 学校の野球部の保護者
- 子供に野球を教えたい親

## サービスの利用イメージ
- 試合観戦中にリアルタイムで用語検索
- 野球ニュースを見ている時
- 野球談議中にわからない用語が出てきた時

### 具体的な使い方の流れ
1. 初回利用時に理解度（初心者/中級者/上級者）を選択
2. 分からない用語を検索ボックスに入力
3. 2-3秒で理解度に合わせた説明が表示
4. 関連用語3つが提案され、さらに学習を深められる
5. 理解が進んだら設定で理解度をアップデート(初心者→中級者)

## サービスの差別化ポイント・推しポイント
AIがユーザーの現在の理解度に合わせて説明を最適化
「一律の野球用語辞典」と異なり、ユーザーの知識レベルに応じて表現を自動調整します。検索結果は2~3秒で表示します
- 初心者「野球以外の例え」を利用し、優しい説明(専門用語なし)
- 中級者 観戦時のポイントや状況説明を追加
- 上級者 チーム背景 戦術的な要素も追加

よく検索される用語はキャッシュ保存で高速表示し、より早く、ユーザーに検索結果を表示
出力時に関連用語3つの提案
単語を調べるだけで、理解を広げるための関連用語も提示。
初心者でも自然に学習が進む設計になっています

## 機能候補
MVPリリース
- レベル選択[ 初心者 ] [ 中級者 ] [ 上級者 ] のボタンを設置し、現段階のユーザーの理解度を確認(初回のみ)
- openAIのAPIを使用し、ユーザーの理解度 (初心者 中級者 上級者) にわけて、結果を出力
- ユーザーの理解度を変更するためのボタンを設置(例 初心者→中級者)
*理解度*
初心者 ほとんどない / ルールもよくわからない
中級者 観戦やゲーム経験はある / 大まかなルールはわかる
上級者 プレー経験あり / 戦術も興味あり

本リリース
- スマホでも見れるようにレスポンシブ対応
- 関連用語3つの提案（継続利用促進）
- APIのコスト削減、高速レスポンスのためにキャッシュ用のテーブルを作成
