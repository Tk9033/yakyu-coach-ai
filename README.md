# yakyu-coach-ai

## サービス概要
野球中継の専門用語に置いていかれたり、子どもの試合で何が起きているか分からない人のためのアプリです。
AIがあなたの理解度に合わせて、専門用語を野球以外の例えや観戦時のポイントを使って分かりやすく解説します。
検索するだけでその場で理解でき、恥ずかしくて聞けない疑問もすぐに解消できます。

## このサービスへの思い・作りたい理由
高校時代、野球部の保護者会に父が参加しましたが、周りは野球経験者や指導者ばかりで、野球のルールしか分からない父は戦術や専門用語の話についていけず、あまり会話に参加できなかったみたいです。当時は「仕方ない」と思っていましたが、自分が社会人になり野球未経験者と話す中で、野球に興味はあるのに理解できず楽しめない人が多いと気づきました。野球観戦は好きだけど、詳しい話はわからない人、子どもが野球を始めて、もっと理解したいと思っている保護者、野球に興味はあるけど、どこから学べばいいかわからない人でも理解度に合わせて学べるアプリがあれば、もっと野球を楽しめるのではと思い、開発を決めました。

## ユーザー層について
### 野球初心者
- パートナーが野球好きで一緒に観戦したい人
- 野球中継の専門用語が分からない方
- 職場の野球談義についていけない方
- 野球部のマネージャー

### 子供が野球を始めた保護者
- 少年野球・部活動の保護者
- 試合をもっと楽しみながら応援したい方
- 子供に野球を教えたい親

## サービスの利用イメージ
- 試合観戦中にリアルタイムで用語検索
- 野球ニュースを見ている時
- 野球談議中にわからない用語が出てきた時

### 具体的な使い方の流れ
1. 初回利用時に理解度（初心者/中級者/上級者）を選択
2. 分からない用語を検索
3. 2-3秒で理解度に合わせた説明が表示
4. 関連用語3つが提案され、自然に学習を促進
5. 理解が進んだら設定で理解度をアップデート(初心者→中級者)

## サービスの差別化ポイント・推しポイント
- AIがユーザーの現在の理解度に合わせて説明を最適化
- 一般的な説明サイトと異なり、説明の難易度が自動で変化
  - 初心者「野球以外の例え」を利用し、優しい説明(専門用語なし)
  - 中級者 観戦時のポイントや状況説明を追加
  - 上級者 チーム背景や戦術的な高度な視点も追加

- キャッシュにより人気用語は高速表示
- 検索するだけで関連用語を3つ提示し、知識が点から線につながる学習体験を提供

## 機能候補
MVPリリース
- 初回の理解度選択 [ 初心者 ] [ 中級者 ] [ 上級者 ]
- openAIのAPIを使用し、ユーザーの理解度 (初心者 中級者 上級者) にわけて、結果を出力
- ユーザーの理解度を変更するためのボタンを設置( 初心者⇄中級者⇄上級者)
*理解度*
初心者 ほとんどない / ルールもよくわからない
中級者 観戦やゲーム経験はある / 大まかなルールはわかる
上級者 プレー経験あり / 戦術も興味あり

本リリース
- スマホでも見れるようにレスポンシブ対応
- 関連用語3つの提案（継続利用促進）
- 人気用語キャッシュ機能
   - DB保存 → 即時表示
  - API呼び出し回数の最適化
